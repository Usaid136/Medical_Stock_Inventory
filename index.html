💊 Medical Store Inventory System

Medical Store Inventory is a responsive PHP & MySQL-based application designed to help pharmacies and medical stores efficiently track and manage their medicine inventory. It features real-time alerts for low stock and expiring products, a category-based UI, and complete CRUD functionality for medicines.

🚀 Features

  • 📦 Add, update, and delete medicine records
  • ⚠️ Auto-alerts for low stock and expiring medicines
  • 📊 Organized view by pharmaceutical categories (tablets, syrups, etc.)
  • 📍 Track location (e.g., Godown 1, In Shop)
  • 🌗 Dark/Light mode toggle
  • 🔍 Live search and dynamic filtering
  • 📅 Tracks retail/wholesale price, expiry date, and date added

🛠️ Tech Stack

  • Frontend: HTML5, CSS3, Bootstrap 5
  • Backend: PHP
  • Database: MySQL
